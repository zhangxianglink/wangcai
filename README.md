+ 关于文件目录的讲解
  1. Public 存放静态文件
  2. src.assets 存放图片
  3. src.componments 存放组件
  4. Src.router 路由
  5. src. views 最外层的页面
  6. main.js 是界面的入口
  7. Test 测试代码
  8. Vue.config.js 相当于webpage 的配置文件。
